# dine-n-wine

## Purpose
A website that allows users to pick a recipe of their chosing, save some favorite recipes, and find out nutritional information about the ingredients

## Built With:
* Html
* CSS
* JavaScript
* Pure CSS (framework)
* Google Fonts
* Edamam API
* TheMealdb API
